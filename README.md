## 🇬🇧 English

### About

**Car Shop** is a handcrafted static website built with pure HTML5 and CSS3 — no frameworks, no dependencies. Each brand gets its own elegant card with history, models, and customer reviews.

### ✨ Highlights

| Feature | Details |
|---|---|
| 🌑 Dark theme | CSS custom properties–based color system |
| 📌 Sticky nav | Header stays visible while scrolling |
| ⚡ Hover animations | Cards lift smoothly on hover |
| 💬 Reviews | Quote-style blocks with blue accent border |
| 📱 Responsive footer | Flexbox layout adapts to any screen |
| 🏷️ Semantic HTML | `<article>`, `<section>`, `<nav>`, `<address>` |

### 🎨 Design Tokens

| Variable | Value | Usage |
|---|---|---|
| `--bg` | `rgb(15, 17, 21)` | Page background |
| `--surface` | `rgb(23, 26, 33)` | Header, footer |
| `--card` | `rgb(31, 36, 46)` | Brand cards |
| `--accent` | `rgb(61, 139, 255)` | Headings, buttons, links |
| `--accent2` | `rgb(46, 204, 113)` | Section headings inside cards |
| `--text` | `rgb(238, 241, 245)` | Body text |
| `--muted` | `rgb(148, 163, 184)` | Secondary text |

**Fonts:** [`Rajdhani`](https://fonts.google.com/specimen/Rajdhani) (headings) · [`Nunito`](https://fonts.google.com/specimen/Nunito) (body)
